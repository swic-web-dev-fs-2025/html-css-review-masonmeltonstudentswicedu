Core Concepts

CSS Selectors

Selectors target elements to style. You’ll use these with querySelector() and Tailwind classes.

/* Element selector */
h1 { color: blue; }

/* Class selector */
.highlight { background: yellow; }

/* ID selector */
#header { font-size: 24px; }

/* Descendant selector */
div p { margin: 1rem; }

/* Child selector */
ul > li { list-style: none; }

/* Attribute selector */
input[type="text"] { border: 1px solid gray; }
