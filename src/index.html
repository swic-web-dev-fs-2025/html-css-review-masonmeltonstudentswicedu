Core Concepts

CSS Selectors

/* Element */ h1 { color: blue; }
/* Class   */ .highlight { background: yellow; }
/* ID      */ #header { font-size: 24px; }
/* Desc    */ div p { margin: 1rem; }
/* Child   */ ul > li { list-style: none; }
/* Attr    */ input[type="text"] { border: 1px solid gray; }

The Box Model

Every element is a box with content, padding, border, and margin.
